Instructions

  • Design slides in the html:
    • Slides go in section class="slide" elements
    • The id attribute is the slide number. Set the total number of slides with the global js variable num_slides.
  • Navigate through the slideshow clicking, shift-clicking, using the left and right arrow keys, or by pressing "g" on the keyboard to jump to a slide.
    • Tip: Hold the alt key to ignore a click.
    Press "b" on the keyboard to jump between the current and last slide.

Use divs for Definitions & Theorems

Here's a theorem:
There are no natural numbers $\natural = (1, 2, 3, \ldots)$ $x$, $y$, and $z$ such that $x^n + y^n = z^n$, in which $n$ is a natural number greater than 2.
A set $C$ is convex if for all $x,y \in C$ and for all $\alpha \in [0,1]$ the point $\alpha x + (1-\alpha) y \in C$.

Images & Videos Are Easy

  • Its easy to insert images/videos
  • You don't even need to give the whole path name. The variable src_dirs lets you set up source directories for tags that you specify.
  • Once you've set up the source directory, just use the filename attribute instead of the src attribute.
